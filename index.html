Talk2Me-MapGen

Cada planeta pode ter uma lista de elementos nele, representado os geradores por quadrados e os consumidores por triângulos, além de ter a cor representando qual joia eles estão relacionados. O jogador em roxo movimenta por si só, tomando decisões a cada 5 segundos a favor de percorrer o menor caminho para coletar e entregar as joias, entretanto não realiza um movimento ótimo de utilizar o menor número de movimentos para conseguir a maior pontuação. Comandos: use as setas para movimentar o mapa, os números entre 0 a 5 servem para trocar de camada de mapa de influencia, enquanto 6 retorna para tela original
